# webgame

## Dependencies

### Client

```sh
cd ./webgame
npm i
npm run format
```

### Server

```sh
cd backend
python -m venv .venv
backend/.venv/bin/pip install -r /path/to/requirements.txt
```

## Run (devel)

### Client

```sh
cd ./webgame
npm run dev
```

### Server

```sh
cd ./backend/app
./.venv/bin/python ./main.py
```
